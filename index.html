Journal

My goals

I am hoping to build the foundation for future success, with this class. I feel like this class will teach me the fundamentals of becoming a life long student of coding! Some of my more concrete goals would be to learn the basics to carry me into the Javascript Immersion class, and learn how to make the best websites possible.

What I don't know!

I have concerns with whether or not I will be able to make it outside of this class, on wether or not my skills and knowledge will be enough to land me a job! There is also a list of the many things I just don't know, yet!

  • HTML
  • CSS
  • Command Line
  • Git
  • Numerous other things to be listed later!

I hate Git

If I were to be honest, I didn't have the problems with Git that I thought I would. To be clear, I was leery and nervous at working with the terminal in general. Luckily I had the time to go through the Treehouse course on using the terminal and Git. Both of those courses prepared me work with the real deal once class started. I was also able to draw on my experience playing text adventure games, such as Zork. Zork is a text adventure game which only has a text interface. Git and the terminal was a trip back to my childhood!

Glossary

This card collects terms and their definitions into a Description List Element (dl) for later reference.

tag name
The word inside an HTML tag that defines the element.
tag
A tag name and opening and closing delimiters. A closing tag puts a forward slash in front of the tag name.
element
An opening tag, some content, and a closing tag. Some elements don't have closing tags and some elements are self closing.

Git, the Savior

Git has ended up being as useful and nifty as I would have hoped! I can't wait to use it further.

Week 3

It's hard to believe that three weeks has already gone by. In those three weeks, I have realized how little I actually know. What a humbling experience it has been! Reflecting back on these past weeks also allows me to see how far I have come. I have now written one website from scratch and used a template to build another one! I am excited to see where the next three months and the next three years brings.
